# Kairo

A Collaborative Text editor Plugin .
Its will as of now operate as Neovim plugin for collaborative text editing. Though the GO[-backend] part will be pluggable to any editor you would like to add collaborative text editing with bit of tinkering.

Its a Open source Project ,so feel free to use or contribute to it.
